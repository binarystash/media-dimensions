Adds width and height columns to the media library

== Description ==

Media Dimensions makes your media files easier to manage by adding sortable width and height columns to the Wordpress media library.

It supports the following file types.

.jpg
.jpeg
.png
.gif
.mp4, .m4v (MPEG-4)
.mov (QuickTime)
.avi
.mpg

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder 'media-dimensions' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Width and height columns should appear in the media library.
